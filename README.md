按下列比例分段累加计算律师费：
1万元及以下收费2500-3000元；
1万（不含）—10万元（含）部分收费比例为6%-9%；
10万-50万元（含）部分为5%-6%；
50万-100万元（含）部分为4%-5%；
100万—500万元（含）部分为3%-4%；
500万元以上部分由律师事务所和委托人协商确定。
